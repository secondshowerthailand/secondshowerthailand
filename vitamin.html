Re:vita Filter วิตามินฟิลเตอร์สำหรับฝักบัว (3 กลิ่น)

วิตามิน C (แอสคอร์บิกแอซิด) 51% + E + B3 + B5 + B7

[ กลิ่นเอกลักษณ์ของ Second Shower ]

กลิ่นเอกลักษณ์ของ Second Shower มีทั้งหมด 3 กลิ่น ได้แก่ สีขาว Fragrance (ไม่มีกลิ่น)
สีเขียว Jeju Fresh Green (กลิ่นสดชื่นธรรมชาติจากเกาะเชจู) สีชมพู Mauritius Flower​ (กลิ่นหอมจากดอกมอริเชียส) กลิ่นวิตามินทั้งหมดของเราปราศจากสารที่ก่อให้เกิดภูมิแพ้ ช่วยให้คุณได้สัมผัสกลิ่นหอมแบบไร้กังวล​

[ Jeju Fresh Green กลิ่นสดชื่นธรรมชาติจากเกาะเชจู ]

กลิ่นหอมของ Jeju Fresh Green มอบความสดชื่นบริสุทธิ์จากธรรมชาติ ผสานกลิ่นของทุ่งหญ้าเขียวขจีและลมเย็น สัมผัสบรรยากาศที่เงียบสงบของเกาะเชจู ให้ความรู้สึกสดชื่นและเงียบสงบเหมือนได้พักผ่อนกลางธรรมชาติ​

[ Mauritius Flower กลิ่นหอมจากมอริเชียส ]

ได้รับแรงบันดาลใจจากทิวทัศน์ของประเทศมอริเชียส เกาะเขตร้อนที่ตั้งอยู่ทางตะวันตกเฉียงใต้ มหาสมุทรอินเดีย รวมทั้งดอกไม้และผลไม้ ตลอดจนดินเจ็ดสีอันเป็นเอกลักษณ์ ที่เกิดจากการระเบิดของภูเขาไฟ​
Top note : แอปริคอต, มะพร้าว ​
Middle note : การ์ดิเนีย, กระดังงา ​
Base note : ไม้ซีดาร์, วานิลลา ​
​

[ Fragrance Free ไม่มีกลิ่น ปราศจากน้ำหอม ]

ปราศจากน้ำหอม ปราศจากสีสังเคราะห์ ที่ทำให้เกิดการระคายเคืองต่อผิว เมื่อไม่มีอโรมาเธอราพี
จะเน้นไปที่การขจัดคลอรีนและคลอรามีนที่ตกค้างออกเพียงอย่างเดียว แนะนำสำหรับสตรีมีครรภ์ เด็ก และผู้ที่ไวต่อกลิ่นหอมหรือผู้ที่แพ้น้ำหอม ​
​

[ วิตามินคอมเพล็กซ์ 5 ชั้น ]

ด้วยคุณสมบัติของวิตามินซี ที่ช่วยฆ่าเชื้อ ดักจับคลอรีนและคลอรามีนที่ตกค้างอยู่ในน้ำได้ถึง 99.9% ยังสามารถช่วยปรับสภาพน้ำ ให้เทียบเท่าระดับน้ำดื่มได้ และปรับสมดุลผิวให้แข็งแรง วิตามินคอมเพล็กซ์ที่ช่วยบำรุง ประกอบด้วย วิตามิน E สำหรับการปกป้องผิว วิตามิน B3 ช่วยบำรุงผิว กระจ่างใส / วิตามิน B5 ช่วยให้ความชุ่มชื้นแก่ผิว และวิตามิน B7 ช่วยบำรุงหนังศีรษะ และช่วยฟื้นฟูผิวให้ดูสุขภาพดี สะอาด​
​​
​- ขจัดสนิมและสิ่งสกปรกได้ถึง 5 ไมครอน
- ขจัดคลอรีนและคลอรามีนที่ตกค้าง 99.9% ด้วยวิตามินซี​
​

*ระยะเวลาการเปลี่ยนไส้กรอง

ช่วงเวลาแนะนำสำหรับการเปลี่ยนไส้กรองคือประมาณหนึ่งเดือน แต่อาจแตกต่างกันไปตามสภาพการใช้งาน ​ หากมีการปนเปื้อนของตะกอนอย่างรุนแรงหรือไส้กรองวิตามินหมด ควรเปลี่ยนไส้กรองใหม่​
